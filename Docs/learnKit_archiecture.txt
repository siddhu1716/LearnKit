        MODE -->|conf >= 0.90| PRESC["PRESCRIPTIVE<br/>follow skill, minimal reasoning"]

        PRESC --> COMP
        GUID --> COMP
        EXPL --> COMP
        COMP["5 · Context Composer<br/>composer.py · layered per type block<br/>1200 token hard cap<br/>skills, failures, facts, prefs, heuristics"]
